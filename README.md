# Task-1-Iris-Dataset-Analysis
Data Science Internship Task 1 - Iris Dataset Analysis

## Objective
Analyze the Iris flower dataset and build a machine learning model to classify iris species based on flower measurements.

## Dataset
Iris Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
1. Data Loading
2. Data Exploration
3. Data Visualization
4. Feature Analysis
5. Classification Modeling
6. Model Evaluation

## Key Findings
- Petal length and petal width are strong predictors of species.
- The dataset is well-balanced across classes.
- Machine learning models achieved high classification accuracy.
